## 📖 快速开始

### 1. 新建患者档案
1. 点击 **"📝 新建患者档案"**
2. 填写患者详细信息
3. 点击 **"💾 保存档案"**
4. 系统自动创建患者专属文件夹

### 2. 生成病程记录
1. 选择患者
2. 选择模板类型
3. 点击 **"✨ 生成病程"** 或 **"📖 从知识库选择"**
4. 根据实际情况修改
5. 点击 **"💾 保存记录"**

### 3. 管理知识库
1. 点击 **"📚 知识库管理"**
2. 选择学科
3. 可以进行以下操作：
   - 点击 **"➕ 新建"** 创建新模板
   - 选择模板后点击 **"✏️ 编辑"** 修改模板
   - 选择模板后点击 **"🗑️ 删除"** 删除模板
   - 点击 **"📊 批量导入"** 从CSV导入
   - 点击 **"📂 导入文件夹"** 批量导入文档
   - 点击 **"🔄 刷新"** 更新列表

### 4. 办理出院
1. 选择患者
2. 点击 **"🏥 办理出院"**
3. 填写出院信息
4. 确认后自动生成Word文档并归档

## 📊 知识库功能详解

### 模板编辑功能
1. 在知识库管理界面选择学科
2. 点击 **"➕ 新建"** 创建新模板，或选择模板后点击 **"✏️ 编辑"**
3. 在编辑界面填写：
   - **标题** - 模板名称（必填）
   - **类型** - 选择模板类型（首次病程/日常病程/出院小结等）
   - **分类** - 选择分类（自动分类）
   - **关键词** - 可选，便于搜索（逗号分隔）
   - **内容** - 模板内容（必填）
4. 点击 **"保存"** 保存模板

### 批量导入功能
#### 准备CSV文件
CSV文件格式示例：
```
标题,类型,分类,关键词,内容
脑梗死首次病程,首次病程,神经系统,脑梗死,患者因"言语不清、右侧肢体无力"入院...
颈椎病日常病程,日常病程,骨关节,颈椎病,患者今日一般情况可，...
```

#### 导入步骤
1. 在知识库管理界面选择学科
2. 点击 **"📊 批量导入"**
3. 选择CSV文件
4. 系统显示导入结果
5. 点击确定完成

#### CSV列说明
| 列名 | 必填 | 说明 |
|------|------|------|
| 标题 | 是 | 模板名称 |
| 类型 | 否 | 模板类型（首次病程/日常病程/出院小结） |
| 分类 | 否 | 分类名称（自动创建） |
| 关键词 | 否 | 搜索关键词（逗号分隔） |
| 内容 | 是 | 模板完整内容 |

### 模板删除功能
1. 在知识库管理界面选择学科
2. 选择要删除的模板
3. 点击 **"🗑️ 删除"**
4. 确认后模板将标记为非活跃状态（保留历史记录）

## 🏥 出院功能

### 出院步骤
1. 选择患者
2. 点击 **"🏥 办理出院"**
3. 填写：
   - 出院日期
   - 出院诊断
   - 出院医嘱
4. 点击 **"🏥 确认出院"**
5. 系统自动：
   - 更新患者状态
   - 保存患者档案
   - 生成Word出院记录
   - 归档所有病程到患者目录

### Word文档结构
- 基本信息（姓名、性别、年龄、床号、住院号）
- 入院诊断和出院诊断
- 主诉、现病史、体格检查、辅助检查
- 诊疗经过
- 病程记录摘要（包含所有病程）
- 出院情况和出院医嘱

## 🚀 编译与部署

### 使用发布脚本
1. 双击 `发布.bat`
2. 等待自动编译完成
3. 在 `publish` 文件夹中找到 `MedicalProgress.App.exe`

### 手动编译
```bash
cd MedicalProgress\src\MedicalProgress.App
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ..\..\publish
```

## 💾 数据管理

### 数据库位置
- **SQLite数据库**：`%LOCALAPPDATA%\MedicalProgress\medical_progress.db`
- **患者目录**：`%USERPROFILE%\Documents\MedicalProgress\Patients\`

### 备份建议
- 定期备份数据库文件
- 患者目录自动保存在文档文件夹
